# Calculate margin of error for weighted proportion

Calculate margin of error for weighted proportion

## Usage

``` r
calc_moe(p, n, deff)
```

## Arguments

- p:

  Proportion (as percentage, e.g., 30 for 30%)

- n:

  Unweighted sample size

- deff:

  Design effect

## Value

Margin of error (as decimal, e.g., 0.0435 for 4.35 percentage points)
